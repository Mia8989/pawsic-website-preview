Gmail

Settings → See all settings → General → Signature → click at the end of your existing signature → press Enter once → paste with Cmd+V (Mac) or Ctrl+V (Windows) → Save Changes.

Outlook (web)

Settings → View all Outlook settings → Mail → Compose and reply → click at the end of your existing signature → press Enter → paste with Cmd+V / Ctrl+V → Save.

Outlook (Mac desktop)

Outlook → Preferences → Signatures → select your signature → click at the end of the text → paste with Cmd+V.

Outlook (Windows desktop)

File → Options → Mail → Signatures → select your signature → click at the end of the text → paste with Ctrl+V → OK.
